# python_bypasser

## Intro 

Web-scrapping is a technology for getting info
from pages in www with scripts also known as web crawlers or bots.

During scrapping web one can face
with a problem of blocking.

Here is an example of bypassing capthca block and try to log in
with given username and password.

Based on https://medium.com/analytics-vidhya/how-to-bypass-recaptcha-v3-with-selenium-python-7e71c1b680fc

## Usage
Run from the command line:
```bash
$ git clone https://github.com/lag-grunge/python_bypasser bypasser
$ cd $_
$ virtualenv --python=(PATH to python bin e.g. /usr/bin/python3.8) venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ python main.py <username> <password>
```
